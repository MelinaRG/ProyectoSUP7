Reglas de seguridad Firebase, estas son extremadamente básicas y no deberían usarse en un proyecto más ambisioso / con mayor alcance

Ya que por ejemplo, cualquier jugador podría borrar todos los elementos del juego o editar lo que sea que deseen en cada nodo "player" individual

```
{
  "rules": {
    ".read": "auth != null",
    ".write": false,
    "players": {
      "$uid": {
        ".write": "auth != null && auth.uid == $uid"
      }
    },
    "coins": {
        ".write": "auth != null"
    }
  }
}
```
